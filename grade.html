Grade

The letter grade for this homework will be broken down as follows:

Letter Requirement
D
Wrangle data into a hierarchical JSON format.
C
The D-level requirements plus implement one node/link diagram with details-on-demand interactivity in D3v5.
B The C-level requirements plus implement one space filling diagram with details-on-demand interactivity in D3v5.
A The B level requirements plus filtering, brushing, or zooming for one of the visualizations.

Details on each grade level.

D-Level Functionality

Successfully wrangle the data into a hierarchical format that can be interpreted by either d3.hierarchy() or d3.stratify(). The hierarchy must have height 3 or more (i.e. at least 3 levels including the root and leaf nodes) and each leaf node must have an attribute.The requirments for this level has been met

C-Level Functionality

Complete the D-level functionality and implement a node link visualization using the d3.cluster() or d3.tree() layout. You may use any Coordinate system (Cartesian or polar) and any edge style (straight or curved).The requirments for this level has been met

B-Level Functionality

Complete the C-level functionality and implement a space-filling or enclosure visualization using d3.partition(), d3.treemap(), or d3.pack() layouts. You may use any Coordinate system (Cartesian or polar).The requirments for this level has been met

A-Level Functionality

Complete the B-level functionality and implement filtering, brushing, or zooming in either of the two hierarchical visualizations you created. You only need to implement this interactivity in one of the visualizations, but must still support details-on-demand interactivity in that visualization.The requirments for this level has been met
